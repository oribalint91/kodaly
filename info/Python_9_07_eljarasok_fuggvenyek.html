🐍 Python alapok: Függvények (Saját parancsok)

11. Függvények: Tanítsunk új szavakat a gépnek!

Eddig a Python beépített parancsait használtuk (mint a print(), input(), len()). De mi van akkor, ha van egy bonyolult feladatunk – például egy karakter sebzésének kiszámítása –, amit a programunkban hússzor is el kell végeznünk? A programozók utálnak gépelni, ezért kitalálták a függvényeket (Functions). A függvény egy általunk írt, elnevezett kódblokk, amit utána egyetlen szóval bármikor újra meghívhatunk!

Az alapok: Hogyan írjunk saját parancsot?

Egy függvény létrehozásához a def (define = definiálás) kulcsszót használjuk. Ezt követi a függvényünk neve, két zárójel, majd a jól ismert kettőspont.

# 1. LÉPÉS: Létrehozzuk (definiáljuk) a függvényt
def koszonto():
    print("-------------------------")
    print("  Üdvözöllek a játékban! ")
    print("-------------------------")

# 2. LÉPÉS: Meghívjuk (használjuk) a függvényt!
# Ezt akárhányszor megtehetjük, anélkül, hogy újra leírnánk a fenti 3 sort.
koszonto()
koszonto()
Fontos szabály: A gép csak "megtanulja" a függvényt a def sornál, de nem futtatja le! Csak akkor hajtja végre a benne lévő kódot, ha utána név szerint meghívod: koszonto().
Paraméterek: A recept hozzávalói

A függvényeink sokkal okosabbak lesznek, ha adatokat is tudunk nekik átadni. Ezeket az adatokat paramétereknek hívjuk, és a zárójelek közé írjuk őket.

# A 'nev' egy paraméter: egy üres doboz, ami várja az adatot
def szemelyes_koszonto(nev):
    print(f"Szia {nev}, jó, hogy újra itt vagy!")

# Amikor meghívjuk, kötelező megadni neki az adatot!
szemelyes_koszonto("Bálint")
szemelyes_koszonto("Anna")

Akár több paramétert is megadhatsz, vesszővel elválasztva: def harc(tamado, vedekezo):

A Mágikus return: Amikor a gép visszaad valamit

Ez a programozás egyik legfontosabb megértendő része! Képzelj el egy turmixgépet (ez a függvény). Beleteszed az epret és a tejet (ezek a paraméterek). A gép összedarálja őket.

  • Ha a függvényedben print() van: A turmixgép megcsinálja a turmixot, de ő maga meg is issza. Te csak nézheted a képernyőn, hogy finom volt.
  • Ha a függvényedben return (visszatérés) van: A turmixgép kiönti neked egy pohárba az eredményt, amit te elmenthetsz egy változóba, és később odaadhatod a barátodnak!
def terulet_szamito(a_oldal, b_oldal):
    eredmeny = a_oldal * b_oldal
    return eredmeny  # Visszaadjuk az értéket a főprogramnak!

# Mivel a függvény RETURN-t használ, az eredményét be tudjuk tenni egy változóba!
szoba_terulete = terulet_szamito(4, 5)
konyha_terulete = terulet_szamito(2, 3)

print(f"A teljes lakás területe: {szoba_terulete + konyha_terulete} négyzetméter.")
Aranyszabály: A return szó azonnal befejezi a függvény futását! Ha a return után írsz még kódot a függvényen belül, azt a számítógép soha nem fogja végrehajtani, mert a return-nél már kiugrott a függvényből.

12. Gyakorló feladatok: A Kódmester receptjei

Nyiss egy új fájlt a VS Code-ban, és próbáld megírni a saját parancsaidat!

1. feladat: A Kikiáltó (Paraméterek gyakorlása)

Írj egy függvényt figyelmeztetes néven, ami egy paramétert vár: egy szöveget (pl. uzenet néven).
A függvény egyszerűen írja ki a képernyőre a kapott szöveget, de tegyen elé három felkiáltójelet!
A program végén (a függvényen kívül) hívd is meg a függvényt valamilyen szöveggel, pl: "Vigyázat, a sárkány felébredt!"

Mutasd a megoldást! 
def figyelmeztetes(uzenet):
    print(f"!!! {uzenet}")

# Főprogram:
figyelmeztetes("Vigyázat, a sárkány felébredt!")
figyelmeztetes("Az életerőd alacsony.")
2. feladat: Az Átváltó-gép (A return használata)

Írj egy euro_valto nevű függvényt, ami egy paramétert vár: a Forint összegét. A függvény ossza el ezt az összeget 400-zal (így megkapjuk az Euró értékét), majd a return segítségével adja vissza az eredményt!
A főprogramban hozz létre egy változót (pl. zsebpenz_euroban), ami egyenlő a meghívott függvény eredményével (adj át neki pl. 8000 forintot). Végül egy printtel írd ki a változó tartalmát!

Mutasd a megoldást! 
def euro_valto(forint):
    euro = forint / 400
    return euro

# Főprogram:
zsebpenz_euroban = euro_valto(8000)
print(f"A zsebpénzem {zsebpenz_euroban} Eurót ér.")
3. feladat (Haladó): A VIP Kidobó (Függvény + Elágazás)

Írj egy beengedhet_e nevű függvényt, ami két paramétert vár: egy nevet (szöveg) és egy életkort (szám).
A függvényen belül írj egy if-else elágazást: Ha a kor 18 vagy annál nagyobb, a függvény térjen vissza (return) egy True (Igaz) logikai értékkel. Ha kisebb, térjen vissza False (Hamis) értékkel!
(Próbáld ki a főprogramban: tedd a függvényhívást egy if feltételbe, és írj ki neki üdvözlést vagy elutasítást!)

Mutasd a megoldást! 
def beengedhet_e(nev, kor):
    if kor >= 18:
        return True
    else:
        return False

# Főprogram:
# Mivel a függvény True-t vagy False-t ad vissza, tökéletes az if mellé!
if beengedhet_e("Péter", 16) == True:
    print("Parancsoljon, fáradjon be!")
else:
    print("Sajnálom, még nem vagy elég idős.")
