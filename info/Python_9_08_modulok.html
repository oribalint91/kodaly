🐍 Python alapok: A "Szerszámosláda" (Modulok)

13. Modulok: Beépített okosságok a Pythonban

Képzeld el, hogy elindulsz túrázni. Nem viszed magaddal az egész garázst, a fűnyírót és a fúrót, csak egy hátizsákot az alapvető dolgokkal (ez a Python alapfelszereltsége, pl. a print és az input). De ha mondjuk hidat akarsz építeni az erdőben, akkor szükséged lesz extra szerszámokra. A Pythonban ezeket az extra szerszámosládákat moduloknak (vagy könyvtáraknak) hívjuk. Ahhoz, hogy kinyissunk egy ilyen ládát, az import (behozatal) parancsot kell használnunk a kódunk legtetején!

A kiszámíthatatlanság mestere: A random modul

A játékok elképzelhetetlenek lennének véletlenszerűség nélkül. A random modul pontosan ezt adja meg nekünk! Nézzük a két leghasznosabb függvényét:

1. Véletlen szám generálása: randint(a, b)
Sorsol egy egész számot a megadott kezdő- és végpont között (a végpontot is beleértve!).

import random  # Ezt mindig a fájl legtetejére írjuk!

gep_szama = random.randint(1, 10)
print(f"A gép dobása: {gep_szama}")

2. Véletlen elem kiválasztása egy listából: choice(lista)
Képzeld el, hogy van egy listád az osztálytársaidról, és a gépnek kell kihúznia a felelőt. Ezt a choice egyetlen sorban megoldja!

import random

fegyverek = ["Kard", "Íj", "Varázspálca", "Gumicsirke"]
kapott_fegyver = random.choice(fegyverek)

print(f"A kincsesládában ezt találtad: {kapott_fegyver}!")
Figyeld meg a szintaxist! Amikor egy modulból használunk egy függvényt, mindig le kell írnunk a modul nevét, utána egy pontot, majd a függvény nevét. (Pl.: random.randint()). Ezzel mondjuk meg a gépnek, hogy "Hé, a randint parancsot a random ládában keresd!"
A matekzseni: A math modul

A Python alapból tud összeadni, kivonni, szorozni és osztani. De mi van, ha mondjuk gyököt akarsz vonni (Pitagorasz-tétel!), vagy szükséged van a Pi (π) pontos értékére a kör területéhez? Irány a math modul!

import math

# Gyökvonás a sqrt (square root) függvénnyel:
gyok = math.sqrt(25)
print(f"25 négyzetgyöke: {gyok}")

# A Pi (π) használata (figyeld meg, itt nincs zárójel a végén, mert ez egy szám, nem függvény!):
kor_sugara = 4
terulet = kor_sugara ** 2 * math.pi
print(f"A {kor_sugara} sugarú kör területe: {terulet}")

14. Gyakorló feladatok: Játsszunk a véletlennel!

Ne felejtsd el az import random vagy import math sorokat betenni a kódod legtetejére!

1. feladat: Dupla kockadobás

Írj egy programot, ami szimulálja két hagyományos (hatoldalú) dobókocka eldobását! Sorsolj ki két számot 1 és 6 között, mentsd el őket változókba, írd ki mindkettőt, majd írd ki az összegüket is!

Mutasd a megoldást! 
import random

kocka1 = random.randint(1, 6)
kocka2 = random.randint(1, 6)

print(f"Az első kocka: {kocka1}, a második kocka: {kocka2}")
print(f"Összesen: {kocka1 + kocka2} mezőt léphetsz.")
2. feladat: Kő, Papír, Olló

Készíts egy listát a három lehetséges lépéssel: ["Kő", "Papír", "Olló"]. A program kérje be a te választásodat egy input() segítségével.
Ezután a gép a random.choice() segítségével válasszon egyet magának a listából, és írja ki!
(Szorgalmi bónusz: a korábban tanult if-elif-else elágazásokkal döntsd is el, ki nyert!)

Mutasd a megoldást! 
import random

lehetosegek = ["Kő", "Papír", "Olló"]

jatekos = input("Mit mutatsz? (Kő/Papír/Olló): ")
gep = random.choice(lehetosegek)

print(f"A te választásod: {jatekos}")
print(f"A gép választása: {gep}")

# Bónusz: Ki nyert? (Részlet)
if jatekos == gep:
    print("Döntetlen!")
elif jatekos == "Kő" and gep == "Olló":
    print("Te nyertél!")
