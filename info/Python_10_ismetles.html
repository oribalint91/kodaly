1. Összetett adatok: Szótárak (Dictionary)

Míg a listákban [] csak egymás után pakoljuk az adatokat, a szótárakban {} Kulcs-Érték párokat használunk. Ez tökéletes egy játékos adatlapjának tárolására!

hos = {
    "név": "Gondalf",
    "életerő": 100,
    "arany": 50
}

# Adat lekérése a kulcs alapján:
print(f"A hős neve: {hos['név']}")

# Érték módosítása (pl. sebződik):
hos["életerő"] = hos["életerő"] - 15

2. Saját parancsok: Függvények (def) és a return

Ha egy kódrészletet többször is használni akarunk (pl. egy támadás kiszámítása), függvénybe csomagoljuk. Emlékszel a turmixgép szabályra? A return adja vissza az eredményt a főprogramnak!

import random

# Függvény létrehozása két paraméterrel
def tamadas(alap_sebzes, fegyver_bonusz):
    # Kiszámoljuk a sebzést egy kis véletlennel megfűszerezve
    vegleges_sebzes = (alap_sebzes + fegyver_bonusz) * random.randint(1, 2)
    return vegleges_sebzes

# Függvény használata: az eredményt egy változóba mentjük
bevitt_talalat = tamadas(10, 5)
print(f"A szörny {bevitt_talalat} sebzést kapott!")

3. A törhetetlen kód: while ciklus + try-except

A felhasználók (játékosok) kiszámíthatatlanok. Ha számot kérsz, tuti betűt írnak be. Ezt a párost használjuk arra, hogy a program ne omoljon össze, hanem újra és újra megkérdezze a játékost, amíg jó adatot nem ad meg.

sikeres_valasztas = False

while sikeres_valasztas == False:
    try:
        # Ha itt betűt ír be, az int() hibát dob, és az except ágra ugrik!
        akcio = int(input("Mit teszel? (1: Támadás, 2: Menekülés) "))
        
        # Ha eljut idáig (nem volt hiba), kiléphet a ciklusból
        sikeres_valasztas = True 
        
    except:
        print("Hiba! Kérlek csak az 1-es vagy 2-es számot üsd le!")

print(f"A választott akciód: {akcio}")
