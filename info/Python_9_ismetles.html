🐍 9. évfolyam: Év eleji Python Gyorstalpaló

Új tanév, új kihívások! Mielőtt belevágunk a komolyabb fejlesztésekbe (függvények, modulok, saját játékok), rázzuk gatyába azt a tudást, amit a korábbiakban megszereztünk. Ebben a leckében egyetlen oldalon áttekintjük a Python legfontosabb alapjait. Ha ezek mennek, bármit meg tudsz építeni!

1. Változók, Adattípusok és Adatbekérés

A változók olyanok, mint a felcímkézett dobozok: adatokat tárolunk bennük. A leggyakoribb típusok: Szöveg (String), Egész szám (Integer), Törtszám (Float) és Logikai (Boolean).

# Adatbekérés és típuskonverzió (szövegből számot csinálunk az int() segítségével)
nev = input("Hogy hívnak? ")
kor = int(input("Hány éves vagy? "))

# Változó értékének növelése
kor = kor + 1

# Kiírás f-string (formázott szöveg) segítségével
print(f"Szia {nev}, jövőre {kor} éves leszel!")

2. Döntéshozatal: Az if - elif - else

A programunk képes útvonalakat választani a feltételek alapján. Ne feledd: a Python a behúzásokból (Tab) tudja, hogy mi tartozik a feltétel belsejébe!

pontszam = 85

if pontszam >= 90:
    print("Jeles!")
elif pontszam >= 70:
    # Ha az első nem volt igaz, de ez igen:
    print("Jó eredmény!")
else:
    # Ha semmi sem volt igaz a fentiek közül:
    print("Ezen még dolgozni kell.")

3. Lista: Több adat egy dobozban

Ahelyett, hogy száz változót hoznánk létre, használhatunk listát []. Ne feledd, a Python nullától (0) kezd el számolni!

hatizsak = ["Kard", "Pajzs", "Alma"]

print(hatizsak[0])  # Kiírja: Kard (a legelső elem)

# Új elem hozzáadása a lista végéhez
hatizsak.append("Kötél")

4. Ciklusok: A fáradhatatlan gép

A for ciklus (Tudjuk, hányszor fut le)

Akkor használjuk, ha egy listán akarunk végigmenni, vagy pontosan tudjuk a lépések számát (pl. range()).

hatizsak = ["Kard", "Pajzs", "Alma"]

# Végigmegy a lista összes elemén egyesével
for targy in hatizsak:
    print(f"A táskámban van: {targy}")
A while ciklus (Addig fut, AMÍG egy feltétel igaz)

Tökéletes játékokhoz, jelszókéréshez, vagy ha nem tudjuk előre a lépések számát. Vigyázz a végtelen ciklusokkal!

jelszo = ""

# Addig kérdezi, amíg a beírt szó NEM egyenlő a titokkal
while jelszo != "nyithataram":
    jelszo = input("Mondd a jelszót: ")
    
print("A barlang ajtaja kinyílt!")
