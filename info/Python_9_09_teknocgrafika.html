🐍 Python alapok: Rajzolás a Teknőccel (Turtle)

15. Teknőcgrafika: A Scratch és a Pencil Code után szabadon!

Emlékszel, amikor Scratch-ben a tollat használva, vagy a Pencil Code-ban JavaScripttel rajzoltattunk alakzatokat? A Python is rendelkezik egy beépített rajzoló modullal, amit turtle-nek (teknőc) hívnak. A logika pontosan ugyanaz: van egy virtuális "robotunk" (a teknőc), akinek parancsokat adhatunk, hogy mennyit menjen előre, és hány fokot forduljon.

Az első vonalak (Alapok)

Ahhoz, hogy rajzolni tudjunk, először be kell importálnunk a turtle modult, és létre kell hoznunk a saját teknőcünket (akit a példában t-nek nevezünk el).

import turtle

# Létrehozzuk a teknőcöt és a rajzvásznat
t = turtle.Turtle()
t.shape("turtle")  # Hogy tényleg teknőc formája legyen!

# Mozgás parancsok
t.forward(100)     # Előre 100 képpontot
t.right(90)        # Jobbra fordul 90 fokot
t.forward(100)

turtle.done()      # Ezt a legvégére írjuk, hogy az ablak ne záródjon be azonnal!
Szabályos alakzatok ciklussal

Ha egy négyzetet szeretnénk rajzolni, nem írjuk le négyszer az előre-fordul parancsokat. Ehelyett – ahogy azt már megtanultuk – bevetjük a for ciklust!

import turtle

t = turtle.Turtle()

# Négyszer ismételjük a behúzott parancsokat
for i in range(4):
    t.forward(100)
    t.right(90)

turtle.done()
Matek zseniknek: Bármilyen szabályos sokszöget rajzolhatsz, ha a 360 fokot elosztod a szögek számával! Például egy hatszögnél: range(6) és t.right(360 / 6).
A Mandala (Ciklus a ciklusban + Random színek!)

Most kössük össze a tudásunkat! Rajzoljunk egy négyzetet, utána forduljunk egy picit, és rajzoljuk meg újra... mindezt 36-szor! Hogy még szebb legyen, a vonalak színét a random modul segítségével minden lépésnél véletlenszerűen választjuk ki egy listából.

import turtle
import random

t = turtle.Turtle()
t.speed(0)  # A 0 a leggyorsabb rajzolási sebesség

szinek = ["red", "blue", "green", "purple", "orange", "magenta"]

# Külső ciklus: 36-szor ismétli a négyzetrajzolást és a forgatást
for i in range(36):
    # Véletlen szín választása a listából
    t.color(random.choice(szinek))
    
    # Belső ciklus: Ez maga a négyzet megrajzolása
    for j in range(4):
        t.forward(100)
        t.right(90)
        
    # A négyzet után fordulunk 10 fokot (36 * 10 = 360 fok, egy teljes kör!)
    t.right(10)

turtle.done()
Varázslat haladóknak: A Fraktálfa (Rekurzió)

A függvényeknél tanultuk a def parancsot. Mi történik, ha egy függvény önmagát hívja meg? Ezt hívják rekurziónak. A természet is így építkezik: egy fa ága kettéágazik, majd azok a kisebb ágak is kettéágaznak, és így tovább...

import turtle

t = turtle.Turtle()
t.left(90)   # Felfelé nézzen a teknőc (a fa törzse miatt)
t.speed(0)

# Saját függvény definiálása
def fa_rajzolas(ag_hossza):
    # Csak akkor rajzolunk, ha az ág még elég nagy
    if ag_hossza > 5:
        t.forward(ag_hossza)      # Megrajzolja a főágat
        t.right(20)                 # Jobbra dől
        
        # ITT A VARÁZSLAT: Meghívja önmagát, de egy rövidebb ággal!
        fa_rajzolas(ag_hossza - 15)
        
        t.left(40)                  # Átlendül a bal oldalra
        fa_rajzolas(ag_hossza - 15) # Bal oldali kisebb ág
        
        t.right(20)                 # Visszaáll középre
        t.backward(ag_hossza)     # Visszamegy az ág tövébe

# Főprogram: Elindítjuk a varázslatot egy 75 hosszú törzzsel!
fa_rajzolas(75)

turtle.done()
Ha ezt lefuttatod, meg fogod látni, milyen gyönyörűen kirajzolja a gép a fa ágait! Próbáld meg átírni a paramétereket (pl. a forgások fokát 20 helyett 30-ra), és figyeld meg, hogyan változik a fa lombkoronája!
